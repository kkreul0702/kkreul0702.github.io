Neural Net study

Attempt to improve the significance of my PhD Thesis

The analysis

The idea is: The matrix element method produces an intermediate results. Can a machine learning algorithm extract more information by combining these with some kinematic variables?

I explored the possibility of using a feed forward neural network to extract more information from the intermediate results generated by the Matrix Element Method (MEM) in conjunction with kinematic variables. I trained a neural network with stochastic gradient descent. Surprisingly, the ROC curves for both methods almost entirely overlap, indicating that the neural network could not improve the performance of the MEM method.
